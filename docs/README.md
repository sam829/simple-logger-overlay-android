# Simple Logger Overlay — Android

A production-ready Android logging library with an **in-app debug overlay** for viewing logs, network requests, and more. Built with Jetpack Compose and Material 3.

---

## What is it?

Simple Logger Overlay gives Android developers a floating debug UI that works without Logcat, USB, or Android Studio. Drop it into any debug build and get:

- A **draggable FAB** that opens a full-screen log viewer
- **Structured logs** with levels, tags, searchable messages, and JSON pretty-printing
- **Network inspection** — every OkHttp request and response captured automatically
- **Session telemetry** — time any user flow, log milestones, attach metadata
- **Export** — share a JSON snapshot of all logs via the Android share sheet
- **Shake-to-open** — open the overlay by shaking the device
